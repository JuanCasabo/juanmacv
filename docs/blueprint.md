# **App Name**: Perfil Profesional

## Core Features:

- Generación de Perfil: Reescritor de perfiles profesionales impulsado por IA, utilizando la experiencia laboral y las habilidades proporcionadas como herramienta. Se centra en el conjunto de habilidades híbridas del usuario, que abarcan la administración, la programación y las ventas.
- Currículum Interactivo: Muestra el historial laboral, las habilidades y la formación académica en un formato interactivo.
- Formulario de Contacto: Permite a los empleadores potenciales contactar directamente con el usuario a través de un formulario de contacto.
- CV Descargable: Ofrece la opción de descargar el currículum vitae como documento PDF.

## Style Guidelines:

- Color primario: Azul suave (#79A8D3), que representa confianza y profesionalidad.
- Color de fondo: Gris claro (#F0F4F7), para dar un aspecto limpio y moderno.
- Color de acento: Naranja cálido (#D38F79), para resaltar los elementos clave y las llamadas a la acción.
- Fuente del título: 'Space Grotesk' sans-serif para una sensación moderna y conocedora de la tecnología. Fuente del cuerpo: 'Inter' sans-serif, que proporciona una contraparte neutral y fácilmente legible para el texto del cuerpo.
- Utilice iconos limpios y profesionales para representar las habilidades y las áreas de especialización.
- Diseño limpio de una sola página con secciones claras para el perfil, la experiencia y las habilidades.
- Animaciones y transiciones sutiles para mejorar la experiencia del usuario sin ser molestas.
